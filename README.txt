# phish_web_app

The phish web app site is a website intended to allow users to signup,
share fish photos, and view these photo submissions randomly while listening 
to music from a Phish playlist. Spotify authentication is required to listen
to full songs. 

(pip install -r requirements.txt) to install all requirements on Windows


Credit for CSS and HTML styling for login, signup, and all password reset and change forms goes to:
	"Login Form Validation in JavaScritp by Dinesh"
